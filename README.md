## About Nexton Accounts

**Nexton Accounts** is a modern accounting management software built with Laravel.

The system is designed to manage business accounting operations efficiently, including company management, financial transactions, reports, and future enterprise-level accounting features.

## Technology Stack

- Laravel Framework 12.x
- PHP 8.2+
- MySQL / MariaDB
- Tailwind CSS
- Vite
- JavaScript

## Current Features

✅ Laravel Authentication  
✅ User Management  
✅ Company Setup  
✅ Database Migration System  
✅ Secure Environment Configuration  
✅ Git Version Control Setup
